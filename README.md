# Around The U.S. - Back-end & Front-end - Sprint 19

Este es el proyecto final que integra un servidor desarrollado en Node.js con Express y una aplicación de frontend construida en React. Permite a los usuarios registrarse, iniciar sesión y compartir fotografías de paisajes.

## Tecnologías utilizadas
- **Backend:** Node.js, Express, MongoDB, Mongoose.
- **Frontend:** React, Vite, CSS (BEM).
- **Seguridad:** JWT (JSON Web Tokens), Bcrypt.js, Celebrate/Joi.
- **Infraestructura:** Google Cloud Platform, Nginx, PM2.

## Enlaces
- **Frontend:** [https://tu-dominio.com](https://tu-dominio.com)
- **Backend (API):** [https://api.tu-dominio.com](https://api.mi-dominio.com)
- **IP del Servidor:** Tu.IP.Publica

## Instalación y ejecución local
1. Clona el repositorio.
2. Ve a la carpeta `backend` y ejecuta `npm install`.
3. Crea un archivo `.env` con tus variables de entorno.
4. Ejecuta `npm run dev` para iniciar el servidor.
5. Ve a la carpeta `frontend`, ejecuta `npm install` y luego `npm run dev`.
